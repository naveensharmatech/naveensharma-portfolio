A working, interactive recreation of the healthcare-SaaS implementation work I did. The Form Builder tab is a real validator with a draw-and-save e-signature pad — type wrong input or leave fields blank and it flags real errors; sign, validate, and submit like a live system.

🔒 Original work, no confidential data. Every screen is built from scratch to illustrate the type of work I performed. It uses invented sample fields, agency names, and people — and contains no real patient information, no client data, and no proprietary employer systems or branding. Built with HTML, CSS, and vanilla JavaScript.
